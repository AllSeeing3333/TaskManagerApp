# TaskManagerApp

Мобильное приложение для управления задачами.

## Функции
- Создание задач с названием, дедлайном, приоритетом и категорией
- Фильтрация задач по категориям (Работа, Учеба, Личное)
- Отметка о выполнении задачи
- Уведомления за 5 минут до дедлайна
- Локальная база данных Room

## Технологии
- Android Studio
- Java
- Room Database
- AlarmManager
- Notifications

## Установка
1. Скачать APK из папки `app/build/outputs/apk/debug/`
2. Установить на телефон Android 7.0+
3. Разрешить уведомления
